How it Works

The Raspberry Pi 4 has built-in Ethernet and WiFi capabilities.


Plugging the WiFi adapter into the USB port on the Raspberry Pi creates 2x WiFi interfaces on the device.


The built-in WiFi adapter will connect to any public WiFi network available, whereas the USB Adapter will broadcast to the user's private network.


This allows the network connection to be accessed via WiFi connection and/or ethernet cable.

Useful for

Having access to a secure private network on-the-go.

When using this device, you no longer need to worry about hackers collecting your personal data via public WiFi network connections.
